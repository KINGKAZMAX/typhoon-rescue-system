# 支援网站

# 一、设计简述

为本次广西受灾情况下罕见病患者家庭以及受灾群众设计的超简化救援申请网站

# 二、 需求背景

## 概况：

依据香港大埔火灾救援网站为原型修正，核心功能为解决因方言所致交流不畅，依托拍照功能进行AI总结现有需求，生成目前待解决问题并散发向各罕见病患者社区以及救援站点

# 四 、功能核心

开发一个简单的操作平台，实现即时救助 

依托照片形式自动筛选分类并生成需求项进行汇总上传

实现拍摄——生成相关救援信息——调配物资——搭配救援方案——联系救援

进行一个超简化高效率的救援

功能核心在于环境分析 数据生成、数据分析、物资储量分析、方案调配协调、救援规划、救援优先级识别、各组织实时对接 

难点在于对接和物资储量的详情

依托现有网站进行开发的情况下 情绪支援服务可酌情增删，优先拍摄和数据生成以及环境识别 次级为实时地图接入、救援联系、物资协调、救援队伍位置、地图引导

罕见病患者优先确认患者体征状态、病情确认、药物储备 罕见病基金会的数据支持与病情分析、救援物资调配、最近救援队伍
