# 广西台风"美莎克"救援平台 · 物资站点 / 安置点模块调研

> **对应 PRD 模块**：物资储备信息 / 物资站点信息 / 救援站点（安置点、物资发放点、医疗点、取水点）。
> **整理时间**：2026 年 7 月 11 日。
> **本轮灾害背景**：2026 年第 10 号台风"美莎克"（强热带风暴）自 2026-07-04 起影响广西，截至 7 月 7 日 20 时，全区 14 个设区市 63 个县区 **37.5 万人受灾、紧急转移安置约 13 万人**；防城港、钦州部分地区降雨超 600mm，南宁横州、贵港西江教育园区出现严重内涝。
>
> **可信度标注约定（与本项目其它文档一致）**：
> - `verified`（已核实）= 已从权威一手来源（政府官网、官方通稿、国家级媒体新华社/央广网/光明网/人民网等）直接确认名称与性质。
> - `media`（媒体报道）= 见于国家级/主流媒体报道，但**具体地址、门牌、容量、电话未逐项在官方名录核到**；名称可信、精确定位需二次核验。
> - `unverified`（待核实）= 仅见搜索聚合 / 二手来源 / 数字有出入 / 无法定位，**上线前必须二次核实**。
> - **安全纪律：本文档绝不杜撰任何地址、门牌、经纬度、电话或容量。** 凡官方通稿只给"场所名称"未给门牌的，地址字段留空或标"以官方现场公示为准"，`verify` 相应下调。数字口径存在多源冲突的（如安置点数量）一律标注冲突并降级。

## 0. 使用与安全须知（务必先读）

1. **安置点具有高度动态性**。台风安置点多为**临时征用**（学校、体育馆、机动车检测站车间、村委会等），开放/关闭随水位与转移进度变化极快，**任何名单都会过期**。平台展示时必须带"更新时间"和"状态"字段，并优先引导用户走**官方实时入口**（第 2 节）而非静态名单。
2. **本轮"美莎克"灾害中，官方通稿普遍只公布"设置了 N 个临时/集中安置点"及个别代表性场所名称，极少逐点公布门牌地址、经纬度、容量、现场电话。** 因此第 1 节的可核实条目多为 `media` 级（名称可信、精确定位待核），能标 `verified` 的极少。
3. **求助/联系仍以三大主线为准**：报警 110、火警/综合救援 119、急救 120（详见 `02-救援电话与组织.md`）。安置点现场一般无对外公开固定电话，联系走属地镇街/村委或 12345。
4. **数字口径以官方最新通稿为准**。同一指标不同媒体/不同时点会有差异（例：贵港集中安置点数量出现"18 个""39 个"两种报道），本文如实标注冲突，不取单一值对外断言。

## 2. 各地"应急避难场所"官方查询途径（**平台应优先接入/外链这些，而非维护静态名单**）

### 2.3 平台落地建议（针对本模块）

- **不要在代码里硬编码"美莎克安置点静态名单"作为唯一数据源**：安置点开关极快，静态名单必然误导。应做成"可运营后台录入 + 官方入口外链 + 更新时间/状态强约束"。
- **精确坐标策略**：官方通稿只给场所名的，`lat/lng` 置空、`geoStatus='name_only'`，前端用"名称 + 城市"做地图弱定位并明确提示"位置未官方核验"，**禁止用第三方地图 POI 猜测门牌当作官方地址展示**。
- **首屏优先展示官方查询入口**（第 2.1 全国公众查询系统 + 属地 12345），把"平台自维护的临时安置点"作为补充并全部带 `verify` 徽标。

## 3. dataModel（物资站点 / 安置点 字段设计）

> 设计目标：覆盖 PRD 的"安置点 / 物资发放点 / 医疗点 / 取水点"，强制携带可信度与时效字段，杜绝把未核实数据当权威数据展示。以下给出字段语义表 + TypeScript 接口 + 一条以真实来源为例的样例记录。

### 3.1 字段语义表

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `id` | string | 是 | 站点唯一 ID |
| `name` | string | 是 | 站点/安置点名称（如"横州市马岭镇振兴村安置点"） |
| `city` | string | 是 | 所属地级市（南宁/钦州/防城港/贵港/北海…） |
| `district` | string | 否 | 区县/乡镇（如"横州市马岭镇"），便于分片包保对接 |
| `address` | string | 否 | 详细地址/门牌。**未官方公布时留空，不得杜撰** |
| `lat` / `lng` | number \| null | 否 | 经纬度。未核验坐标置 null，配合 `geoStatus` |
| `geoStatus` | enum | 是 | `official`（官方给坐标/门牌）\| `name_only`（仅有名称，坐标为弱定位）\| `unknown` |
| `type` | enum | 是 | `shelter`(安置点) \| `supply`(物资发放) \| `medical`(医疗点) \| `water`(取水点)。可为数组以支持"安置+发放"复合点 |
| `capacity` | number \| null | 否 | 可容纳人数。无官方数字置 null，不估算 |
| `occupancy` | number \| null | 否 | 当前在住/使用人数（如有运营回填） |
| `supplies` | Supply[] | 否 | 可提供/库存物资清单（见下）。安置点常见：饮用水、方便食品、被褥、折叠床、应急照明；医疗点：常用药、急救；取水点：饮用水 |
| `contact` | Contact | 否 | 联系方式（电话/单位/属地镇街）。**无对外公开电话时不编造，可填属地 12345 或包保单位** |
| `status` | enum | 是 | `open`(开放) \| `full`(已满) \| `closing`(准备关闭) \| `closed`(已关闭) \| `planned`(拟设) \| `unknown` |
| `serviceScope` | string | 否 | 服务范围/接收对象（如"接收西江教育园区转出师生""周边低洼村屯居民"），罕见病/特殊人群备注可挂此 |
| `accessNote` | string | 否 | 到达/无障碍/道路是否中断提示（灾时关键） |
| `source` | Source[] | 是 | 来源数组：官方通稿/媒体链接/后台录入人，用于回溯与复核 |
| `verify` | enum | 是 | `verified` \| `media` \| `unverified`（含义见文首约定），前端据此显示可信度徽标并决定是否允许导航/拨号 |
| `updatedAt` | string(ISO8601) | 是 | 最后更新时间。前端超过阈值（如 6–12h）应显示"信息可能已过期，请拨 12345 核实" |
| `createdAt` | string(ISO8601) | 是 | 创建时间 |

嵌套结构：

| 结构 | 字段 |
|---|---|
| `Supply` | `item`(名称，如"饮用水") · `unit`(单位) · `quantity`(数量, number\|null) · `stockStatus`(`sufficient`/`limited`/`out`/`unknown`) |
| `Contact` | `phone`(string\|null，禁编造) · `org`(负责单位/镇街) · `is12345Fallback`(bool，标记是否为兜底热线) |
| `Source` | `label` · `url` · `publishedAt` · `sourceLevel`(`official`/`state_media`/`local_media`/`internal`) |

### 3.2 TypeScript 接口

```ts
type StationType = 'shelter' | 'supply' | 'medical' | 'water';
type Verify = 'verified' | 'media' | 'unverified';
type StationStatus = 'open' | 'full' | 'closing' | 'closed' | 'planned' | 'unknown';
type GeoStatus = 'official' | 'name_only' | 'unknown';

interface Supply {
  item: string;                 // 饮用水 / 方便食品 / 折叠床 / 被褥 / 应急照明 / 常用药
  unit?: string;
  quantity?: number | null;     // 无准数则 null，不估算
  stockStatus: 'sufficient' | 'limited' | 'out' | 'unknown';
}

interface Contact {
  phone?: string | null;        // 无对外公开电话时 null，严禁杜撰
  org?: string;                 // 负责单位 / 属地镇街 / 分片包保组
  is12345Fallback?: boolean;    // true 表示这是 12345 兜底而非现场直线
}

interface Source {
  label: string;
  url?: string;
  publishedAt?: string;         // ISO8601
  sourceLevel: 'official' | 'state_media' | 'local_media' | 'internal';
}

interface ReliefStation {
  id: string;
  name: string;
  city: string;
  district?: string;
  address?: string;             // 未公布则空，禁编造
  lat?: number | null;
  lng?: number | null;
  geoStatus: GeoStatus;
  type: StationType[];          // 支持复合，如 ['shelter','supply']
  capacity?: number | null;
  occupancy?: number | null;
  supplies?: Supply[];
  contact?: Contact;
  status: StationStatus;
  serviceScope?: string;
  accessNote?: string;
  source: Source[];
  verify: Verify;
  updatedAt: string;            // ISO8601，前端据此判过期
  createdAt: string;
}
```
