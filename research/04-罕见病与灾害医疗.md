## 二、高危依赖治疗人群 · 灾时（停电 / 断药 / 交通中断）应急操作要点

> 定位：这是给**患者/家属**看的"灾时怎么办"可操作卡片，也是产品内容底稿。
> **总原则**：① 危及生命先打 120；② 任何"能不能停/能不能替代"的判断，尽量先联系原主治医生/原治疗机构；③ 提前储备"应急包"（药品清单、诊断证明、医保卡、近期病历、原厂说明书、便携冷藏、备用电源/氧气）。
> 本节临床要点来源为 FDA、美国 CDC、美国国家肾脏基金会（NKF）、NIDDK、美国肺脏协会（ALA）及中国 CDC《全国自然灾害卫生应急工作指南》等权威机构，URL 见"来源"。剂量/个体化决策仍以主治医生为准。

### 2.1 胰岛素依赖型糖尿病（停电 / 冷藏中断）

**关键事实（FDA / CDC，✅ verified 数值）**
- 正常冷藏：约 **2–8℃（36–46℉）**，未开封在此条件下可存到有效期。
- **应急可脱离冷藏**：厂家原装瓶/笔芯（无论开封与否）可在 **15–30℃（59–86℉）** 室温下存放**最多 28 天**仍有效。
- **绝对不能冷冻**：冻过的胰岛素结构破坏、失效，**不可再用**。
- 高温预警：暴露于 >46℉（约 8℃）超过约 12 小时，效价可损失约 15%（不同资料数据，供理解"越热越快坏"）。现实中若不得不用曾 >30℃ 存放的胰岛素，**要更勤测血糖**。
- **失效识别**：液体变色、出现絮状物/结晶/团块 → 不要用。
- 灾后恢复常态后：把曾长期室温/过高过低存放的胰岛素**丢弃**、更换新药。

**灾时可操作步骤**
1. **降温而非冷冻**：放入带冰的保温箱/小冷藏包，**药品不要直接接触冰块**；或用户手边办法——密封好的药盒放进抽水马桶水箱，可比环境低约 8–11℃、维持 8–12 小时（应急过渡）。
2. **长停电（>12h）**：优先用便携电池医用冷藏盒；无条件则按"室温 28 天"上限使用并勤测血糖。
3. 备温度计监测存放温度；随身留一份不同品牌胰岛素的换用注意（FDA 有"紧急情况下不同胰岛素产品间如何过渡"的说明，换用务必咨询医生/药师）。
4. **危险信号立即就医/打 120**：反复呕吐、意识模糊、深大呼吸、脱水（酮症酸中毒征象）或严重低血糖昏迷。

### 2.2 需血液透析的肾病（透析中断）

**核心（NKF / NIDDK，✅ verified 要点）**
- **预警期先"提前透析"**：台风/洪水来临前，尽量**按医嘱提前做一次透析**——研究显示灾前提前透析者更少需要急救/住院。
- **中断期立刻收紧饮食**（控制钾、磷、钠、液体，防致命性高钾血症与容量超负荷）。NKF 应急"三日餐"示例量级：每日约蛋白 40–50 g、钠 1500 mg、钾 1500 mg、**液体 < 500 cc（约 16 oz）**。
  - **少喝水**：应急指引建议每天液体不超过约 2 杯。
  - **避开高钾食物**：土豆、香蕉、番茄、橙子等；避难所发的餐往往高钠高钾，**尽量自带合规食物**。
- **备三天非易腐、低蛋白/低钾/低钠/低液体的应急食物**。
- **降钾药**：无法及时透析者，临床上可用**降钾树脂（聚苯乙烯磺酸钠）**帮助防止致命高钾——**须由医生指导使用**，不可自行滥用。

**灾时可操作步骤**
1. 立即联系原透析中心确认是否运营、能否加排；不通则找"最近仍在运转的透析点"（查询途径见第三节，或打 12320 / 12345 协调）。
2. 立刻执行应急限钾限水饮食，记录尿量与体重变化。
3. **危险信号立即打 120**：明显气促/不能平卧（肺水肿）、心悸/脉律不齐、四肢无力麻木、胸闷（高钾/容量超负荷征象）。
4. 随身带：透析病历、干体重、用药单、透析方案、医保卡。

### 2.3 需持续供氧 / 呼吸机（停电）

**核心（美国肺脏协会等，🟡 综合权威科普，要点可靠）**
- **呼吸机最危险**：不能容忍长时间断电；家用制氧机约需 300–600 W，呼吸机约 50–300 W。
- **停电即刻切到备用氧气瓶/便携氧**：备用钢瓶要**常年放在随手可及处并保持充满**，事先练熟"从制氧机切到便携系统"的操作。
- **备用电力**：医用 UPS/发电机/外置电池。内置电池一般 30 min–2 h，外置电池组常 4–8 h；便携制氧机电池要常充满并备多块、定期测试。
- **提前登记"生命支持设备用户"**：向供电公司报备可进入"医疗优先/用电保障名单"（国内对应向当地供电营业厅/社区报备重症在家用电设备）。
- **应急包**：手摇/电池收音机、充满电的手机与充电宝、医生与设备供应商联系方式、所有依赖设备清单、至少 **24–48 小时**备用氧。

**灾时可操作步骤**
1. 停电→立即切换备用氧；同时联系原供氧/设备供应商与社区，安排续瓶或转移到有电场所（医院/避难点/亲友）。
2. **氧气瓶获取**：向原家用医疗器械/供氧公司、就近医院氧疗科/呼吸科、120 求助；用电用氧同时告急时优先保呼吸机供电、以备用氧过渡。
3. **危险信号立即打 120**：口唇发绀、明显气促、意识改变、血氧持续下降。
4. 安全提示：用氧环境**严禁明火/吸烟**，远离炉灶、蜡烛（灾时更易失火）。

### 2.4 癫痫 / 需冷藏药物（断药 / 冷藏中断）

**癫痫核心（临床共识，✅ 要点可靠）**
- **绝对不要突然停用抗癫痫药**：骤停可诱发癫痫大发作乃至**癫痫持续状态（status epilepticus，危及生命）**。任何减停都必须医生指导、缓慢进行。
- 灾时**优先保证不断药**：随身多备几日量、带药盒与处方/病历；断药风险高时尽快联系医生/找就近可开药机构或走长期处方。
- **癫痫持续状态（一次发作 >5 分钟，或连续发作间意识不恢复）→ 立即打 120。**

**冷藏药物通用（FDA / CDC，✅ 数值可靠）**
- 需冷藏的还有：部分**胰岛素**（见 2.1）、**肾上腺素笔（epinephrine）**、部分生物制剂/酶替代治疗药、某些疫苗类等。
- **肾上腺素笔**：多数说明书建议约 **15–30℃（59–86℉）** 存放；过冷/过热都降效；勿冷冻。CDC 建议即使未到印刷有效期也每 12–18 个月更换。
- **通用降温法**：冰盒/保温包 + 冰袋（**药品不直接触冰、勿冻**）；短时可用马桶水箱法（低约 8–11℃、8–12 h）；长停电用电池医用冷藏盒；全程用温度计监测；变色/沉淀/异常即弃用。
- **换药/替代**务必咨询医生或药师，不要自行改剂量或换品种。

### 2.5 化疗中 / 免疫缺陷（灾后感染高风险）

**核心（CDC / 肿瘤中心科普，✅ 要点可靠）**
- 化疗患者常有白细胞/血小板低下，灾后**污水、淤泥、霉菌、拥挤避难所**是"感染雷区"：洪灾后可增加钩端螺旋体、甲肝、伤寒、真菌（曲霉等）、军团菌、革兰阴性菌等风险。
- **第一步：联系肿瘤/主治团队**取得个体化指引（能否延后疗程、如何防感染、是否需预防用药）。
- 防护清单：手卫生（免洗洗手液/肥皂）、口罩、手套、湿巾；只用**清洁水**洗漱清创，伤口保持包扎干燥；避免高暴露环境，保持社交距离。
- 免疫缺陷者：可事先向社区/民政了解**"特殊需求避难安置"**，避免与人群高密度混住。
- **危险信号立即就医/打 120**：发热（尤其中性粒细胞减少期发热是急症）、寒战、出血不止、明显乏力/意识改变。

## 三、灾后如何快速找到"仍在运转"的医院 / 透析点 / 药房——官方查询途径

> 原则：**优先走官方/权威渠道核实"是否仍在运营/能否接诊"，别凭网传名单跑空或跑险。** 号码只展示全国统一短号与官方门户。

### 3.1 人工兜底（灾时最可靠，先打电话再跑）
- **12345**（政务便民热线，✅ 全国统一）：找就近仍运营的医院/透析点/药店、协调转运、反映断药断医。
- **12320**（卫生健康热线，✅ 全国统一短号）：寻医问药、心理援助、突发公共卫生事件咨询与投诉。广西已开通。
- **120**（✅）：危及生命直接呼救，并可询问就近可收治机构。
- 直接**电话原就诊/透析/供氧机构**确认是否运营、能否加排、是否转移点位。

### 3.2 官方网上信息源
- **广西卫健委官网** https://wsjkw.gxzf.gov.cn/ （✅）：灾后常发布"开诊医院/停诊公告/急救定点/发热门诊/血透接续安排"等。
- **广西疾控中心** https://www.gxcdc.org.cn/ （✅）：灾后饮水/传染病/防病与市级疾控通讯录。
- **各市卫健委官网**（如柳州 http://wjw.liuzhou.gov.cn/ 等，✅ 政府门户）：本地开诊/药械保障公告。
- **国家医保服务平台 / 广西医保局** http://ybj.gxzf.gov.cn/ （✅ 官方）：查**医保定点医疗机构与定点零售药店**（找仍能刷医保买药、开长期处方的药房）。
- **CHARD 云平台"找医院/找新药/找政策"** https://www.chard.org.cn/ 与病痛挑战"找医院/找组织/找新药/找政策" https://www.chinaicf.org/ （✅）：按**罕见病病种**定位有诊疗能力的医院与药物可及信息（非实时开诊状态，需再电话确认）。

### 3.3 透析点专项
- 血透属"不能断"的刚需：**首选原透析中心**（提前电话确认灾时排班/加排）；
- 原中心停摆 → 打 12345 / 12320 或向广西卫健委公告查"就近仍开放的血透室"；危急走 120 转有血透能力的三级医院急诊。
- 广西罕见病/综合诊疗能力较强、且具透析条件的大型三甲可作为区域兜底方向（见 3.4 名单中的省级医院），但**是否灾时开放仍须电话核实**。

### 3.4 广西罕见病诊疗协作网医院（官方名单，✅ verified）

来源：**《自治区卫生健康委 自治区中医药局关于公布广西罕见病协作网医院诊疗信息的通知》（桂卫医发〔2024〕11 号，2024-04-30 成文）**，广西壮族自治区中医药管理局官网原文逐字核到：http://zyyj.gxzf.gov.cn/xwdt/gxgg/t18385462.shtml

- **广西医科大学第一附属医院**（省内罕见病诊疗牵头/主力，诊疗病种最全，含地贫重型、血友病、原发性免疫缺陷、脊髓性肌萎缩、法布雷病、糖原累积病、Dravet 综合征等 140+ 病种）

> 产品用法：
> ① 罕见病患者"确诊/复诊/开具协作网诊断证明"（申请全国医疗援助工程的前置条件）应优先导向**广西医科大学第一附属医院、广西壮族自治区人民医院**等省级主力；
> ② 这些三甲同时具备透析、ICU、供氧、化疗等高危依赖治疗能力，可作为灾后区域医疗兜底方向；
> ③ **但"灾时是否开诊/能否接诊"必须先电话或看官方公告确认**，名单仅代表平时诊疗能力，不代表实时可用。

## 四、产品化落地建议（给本平台）

1. **"罕见病患者家庭"专用求助流**：入口三级——① 危及生命 → 120；② 找钱/找药/找资源 → 病痛挑战"罕见病医疗援助工程"（400-040-8772，官网入口）；③ 找诊断/找协作网医院 → 广西协作网名单（3.4）。
2. **所有电话按 verify 分级渲染**：✅ 直接可拨；🟡 显示"以官网为准"并提供官网跳转；⛔ 不出现号码，只给官网/邮箱。
3. **"我的应急包"清单模板**：分病种（胰岛素/透析/氧疗/癫痫·冷藏药/化疗）自动生成，含"备药量、诊断证明、医保卡、病历、备用电源/氧气、便携冷藏、主治医生电话"。
4. **灾时医疗资源"先电话确认再导航"**：任何医院/透析点/药房卡片都必须带"拨打确认是否开诊"按钮，避免网传名单误导跑空。
5. **号码可信度巡检**：所有对外号码定期回官网复核（尤其 400 分机号、地方专线），发现不一致以官方为准并降级标注。
